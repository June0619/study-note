# Week 3 (2023-02-20 ~ 2023-02-26)

## 알고리즘 1강 (KNOU)

### 개요
- 좋은 프로그램 = 적절한 자료구조 + 효율적인 알고리즘
- 모든 자료구조는 배열 혹은 연결리스트 중 하나를 사용함

### 선형 자료구조
- Stack (LIFO)
- Queue (FIFO)
- 배열 (물리적 연속)
- 연결 리스트 (논리적 연속)

### 비선형 자료구조
#### 트리 
- 리프노드 (단말노드)
- 자식노드 (비단말노드)
- 이진트리 (Binary Tree)

#### 그래프
- 정점과 간선으로 이루어짐
- 방향 그래프 (1,2) = (2,1) / 무방향 그래프 (<1,2> != <2,1>)
- 가중 그래프 (간선의 가중치)
- 구현 방법
    - 인접 행렬
    - 인접 리스트
